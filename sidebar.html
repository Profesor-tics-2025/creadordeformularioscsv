Información del formulario

Datos del alumno Añade al inicio campos obligatorios de Nombre, Apellidos y Grupo.
▶ ¿Cómo debe estar la hoja de cálculo?

La primera fila debe contener los encabezados. El script los detecta de forma flexible:

ColumnaNombres aceptados
PreguntaPregunta, Titulo, Question*
TipoTipo, Type, Formato
OpcionesOpción1, Opcion2, Opt3…
CorrectaCorrecta, Respuesta Correcta, Answer
PuntosPuntos, Puntuacion, Points, Score
Feedback ✓Feedback Correcto, Comentario Acierto
Feedback ✗Feedback Incorrecto, Comentario Fallo

Tipos válidos: test (por defecto), casillas / multiple, texto / abierta.
Para casillas, separa las respuestas correctas con comas.

Estructura del cuestionario

Pregunta por página Inserta un salto de sección entre preguntas. Ideal para exámenes largos.
Barajar opciones El orden de las respuestas (A, B, C…) varía en cada alumno.
Preguntas obligatorias Obliga a responder todas las preguntas antes de enviar.

Puntuación

Se aplica a las preguntas donde la columna «Puntos» esté vacía.

Barra de progreso

Mostrar progreso Muestra al alumno en qué porcentaje del formulario se encuentra.

ℹ️ Sobre estilos visuales

La API de Google Forms no permite cambiar el color o tema del formulario de forma programática. Puedes personalizarlo manualmente desde el editor de Forms una vez generado.
